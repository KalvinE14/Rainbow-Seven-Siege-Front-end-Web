QUARANTINE MODE
DESCRIPTION
During this Corona pandemic, we provide a new game mode that you can enjoy, it is quarantine mode. We already prepared a special new map, especially for this new game mode. The objective of the operator sided with terrorists is to get out of the room, while the task of operators sided with the counter-terrorist is not to allow the terrorist stronghold to leave the room. There are some side quests you can enjoy too while you are in-game.
